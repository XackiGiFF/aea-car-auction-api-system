# Backend Scripts

Вспомогательные скрипты для запуска и обслуживания backend-системы.

## Файлы

- `compose.sh` единая оболочка запуска compose (`dev|prod`)

## Пример

- `./scripts/compose.sh dev up -d --build`
- `./scripts/compose.sh prod pull`
