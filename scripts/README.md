# Root Scripts

Скрипты верхнего уровня репозитория.

## Файлы

- `bump_wp_plugin_version.sh` увеличивает версию WordPress-плагина `wp-car-auction-plugin-lite`

## Использование

```bash
./scripts/bump_wp_plugin_version.sh patch
./scripts/bump_wp_plugin_version.sh minor
./scripts/bump_wp_plugin_version.sh major
```
